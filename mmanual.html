蝕刻機

  1. 將樣品裝入:


    2. 1.從機體上解下目標靶

    2.將樣品黏在銅膠上後，黏在目標靶的位置偏標誌方格中偏上方

    3.對準標誌線，裝回機體


  2. 蝕刻環境準備:


    3. 1.打開冰水機

    2.打開機體開關(在底下黑色開關)

    3.打開粗抽開關(右邊紅色開關)

    4.打開Ar鋼瓶總開關，順時針為加壓，逆時針為減壓

    5.慢慢調整鋼瓶的微開關，直到Ar輸出的聲音聽不太清楚後接上

    6.打開上方腔體空間，關閉turbo空間，讓機械幫浦抽氣

    7.關閉上方腔體空間，打開turbo空間，讓機械幫浦抽氣

    8.調整兩個腔體的壓力至差不多2e-2torr，打開兩個腔體讓氣壓持續抽到約5e-2torr

    9.打開turbo開關，按下start鈕，讓葉扇轉動頻率提升到963Hz

    10.關閉腔體上方空間，打開腔體下方空間(轉到底再迴轉半圈)

    11.參考之前蝕刻環境的參數，調整Ar輸入大小，控制氣壓

    12.若氣壓不穩定，可微量增加Ar的氣壓，但不要大於一格，否則氣壓太大會損壞turbo的葉扇


  3. 蝕刻機:


    4. 1.調整好氣壓後，打開蝕刻源

    2.參考先前的設定，按下Select Recipe選擇相對應的recipe，按下Enter確認

    3.按下Source待其數值穩定，若無法工作，則微量增加Ar氣鋼瓶的的輸出量，調整至預定氣壓環境再按Source

    4.按下Beam，待其達到紀錄本上的Beam Voltage即可


  4. 開始蝕刻:


    5. 1.將閥門轉90度，蝕刻1分鐘，並記錄蝕刻機參數和氣壓

    2.接著將閥門轉回去，並按下Beam，待值穩定後，再按下Source，等全部的值歸零，關掉蝕刻機

    3.讓蝕刻機休息10分鐘，才可接著蝕刻

    附註:蝕刻的時間和蝕刻機休息的時間可調整，但不可讓蝕刻機工作太久

  5. 結束蝕刻:


    • 無液態氮:


      • 1.關閉下方腔體空間

      2.按下Turbo的Stop鈕(非電源)，靜置半小時，待其降速 3.關掉turbo電源turbo空間

      4.關掉Ar鋼瓶，關掉粗抽開關，關掉機體總電源


    • 有液態氮:


      • 1.用導管接空壓機將氮氣趕出2個小時，同時等下方腔體空間回溫約半個小時後關閉下方腔體空間

      2.按下Turbo的Stop鈕(非電源)

      3.關掉turbo電源，關閉turbo空間

      4.關掉Ar鋼瓶，關掉粗抽開關，關掉機體總電源


鍍金

  1. 裝樣品和清腔體:


    2. 1.將樣品放在旋轉台上，確認樣品不會被靶遮住

    2.關閉腔體

    3.打開機械幫浦，將腔體氣壓抽至2e-2 torr

    4.打開Ar鋼瓶，將Ar通入腔體至0.8 torr

    5.關閉Ar通入腔體

    6.再做一次第四步和第五步


  2. 開始鍍金:


    3. 1.通入Ar至腔體，控制器壓到7e-2 torr

    2.打開冰水機，旋轉台和電漿機(MDX-500)

    3.調整電漿機功率至目標功率加2W

    4.按Start啟動電漿機，鍍目標時間


  3. 結束鍍金:


    4. 1.按Stop關閉電漿機，並關閉機體總開關

    2.關掉旋轉台

    3.關蝴蝶閥，停止Ar通入腔體

    4.關機械幫浦

    5.解除真空，拿樣品出來

    6.關閉腔體，打開機械幫浦，將腔內氣壓抽至2e-2 torr

    7.關Ar鋼瓶


高溫爐退火

  1. 裝樣品:


    2. 1.打開高溫爐腔體

    2.將樣品放在適當大小的陶瓷片上

    3.將陶瓷片和樣品放入高溫爐腔體中，並關閉腔體


  2. 環境設定:


    3. 1.打開冰水機，並關閉冷氣，避免高溫爐受冷氣影響破損

    2.將高溫爐的插座(220V)插上

    3.調整溫度控器至MAN和HOLD檔位

    4.按左下角折角圖案的按鈕，選定ProG LiSt

    5.按返回鍵，調整退火條件(有很多參數要輸入，如rmP.U、dwL.U、SEG.n、tGt、rAtE)

    6.按返回鍵，條回ProG LiSt

    7.按左下角折角圖案的按鈕，調至顯示版面皆為數字(數字代表溫度)


  3. 退火開始:


    4. 1.調整溫度控器至AUTO和RUN檔位

    2.按下電流表下方的紅色按鈕，開始退火

    3.等溫度控制器的溫度降至常溫，即可取出樣品

    4.關閉冰水機


雙槍磁控電漿濺鍍系統(IGZO)

  1. 環境設定:


    2. 1.開冰水機(紅開關→綠開關)

    2.開Ar鋼瓶，大的是鋼瓶壓力(實驗結束後要關，壓力閥會慢慢降下來)。小的是管子壓力(1~2之間，調至穩定狀態)

    3.將真空閥轉45度，洩氣後打開腔體。

    4.把基底電極黏起來，以板子墊高基板，將基板置於中央

    5.開機台總開關，接著打開粗抽開關(記得再次確認洩氣閥有無關緊)

    6.轉下方黑色旋鈕置底，並迴轉兩圈。待氣壓抽至穩定值時，打開Ar的閥(會有"ka"的聲音)

    7.清腔體兩次，若過久沒清則多清幾次

    • 逆時針旋轉閥通入Ar至2e-1 torr。若氣壓下降太慢，則檢查鋼瓶壓力計



  2. 開始鍍:


    3. 1.將220V的插頭(線為棕色，千萬不能折到)接上RF電漿產生機

    2.在2e-1 torr下，打開RF電漿產生機。將值轉到30W，按下"ON"，上面那台會發出聲音。

    3.確認腔體中變為粉紫色時，將功率慢慢調80W，靜置3分鐘，已清理靶材

    4.將電漿功率調製100W，轉開擋板，計時10分鐘


  3. 結束:


    4. 1.將RF電漿產生機的功率降至30W，按下"OFF"，再轉至"0W"，接這關閉電源

    2.先降溫，關Ar氣，關鋼瓶(抽髒東西)

    3.下方黑色旋鈕先關，再洩氣拿樣品

    4.腔體保持真空，拿完樣品後，蓋上蓋子，檢查洩氣閥關好，再打開黑色旋鈕


  4. 退火:


    5. 1.清腔體兩次

    2.通Ar至2e-1 torr

    3.按下"SET/ENT"鈕至"PrG"，按"▲"至"1"，調整參數

    • SP1:
      • 從25至預定溫度

    • tm1:
      • 從25至所需時間

    • SP2:
      • 調至維持溫度

    • tm2:
      • 設定維持時間

    • SP3:
      • 調至25

    • tm3:
      • 調至25

    • 長按▼，畫面顯示RUN



超導(YBCO)鍍膜

    儀器極耗電，盡量不要同時與其他儀器使用
  1. 裝溫控計:


    2. 1.由右至左的接頭依序為"電源線"、"LOAD(加熱器)、"溫度計"

    2.接上電源線(110V)的時候，不可插上插頭，會燒掉

    3.左邊屬來第三條線為接地線

    4.腔體上方接的管線通入氧氣用、側邊則為通入氬氧

  2. 裝樣品:


    3. 1.將腔體慢慢洩氣後，打開腔體

    2.將樣品裝在載台的凹槽上

    3.確認溫度計有插入，並調整載台位置，不要被旋轉台卡到

  3. 升溫階段:


    4. 1.打開冰水機，關閉腔體，將抽氣閥旋開，清腔體，抽到3e-2 torr(小鍍膜狀態一個order)

    2.設定溫控計(升溫速度:20度C、加熱至640度C、Pd持續時間設定2min)

    3.Pr2設定跑程式第二階段(溫度640度C，Pd持續時間設定0min、降溫速度:10度C，到350度C，降溫速度15度C到0度C)

    4.打開旋轉台(旋轉速度不要太快)

    5.若以上程序都設定完成，則只要直接按"RUN"就好，約等30分鐘

  4. 清腔體和靶材:


    5. 1.按溫控計的hold。同時通入氬氧進行清腔體，共兩次

    2.按"手符號"跑出man，調OP(燈絲功率)54，若設定的OP太大則每5調一次

    3.接上電漿產生器的線至腔體，逆時針旋轉旋鈕避免一開始輸出功率過高

    4.接上電漿機的電源線(220V)，打開電源，確認風扇在轉，按綠色鈕開啟，確認腔體電漿電極呈紫色。

    4.調整電漿產生機，轉到65，調整使反射變零

    5.按F3匹配其電容電感值，按F1和F2去調整TUNE和LOAD。若值調不下去可先打開擋板，讓電漿先點起來

    6.通入氬氧，使氣壓維持250 mtorr，靜至10分鐘

  5. 開始鍍膜:


    6. 1.將電漿產生機轉到72

    2.由腔體窗口往內看，轉開擋板，靜至4小時(依需要厚度增減時間，目前為平均速率0.5nm/min)

    3.期間注意氣壓維持250 mtorr，反射為零

  6. 退火:


    7. 1.蓋上擋板，關閉旋轉台，紀錄冰水機溫度，慢慢將電漿產生機的功率調低

    2.關閉氬氧的閥，讓機械幫浦抽氣

    3.打開氧氣鋼瓶，打開氧氣閥，清兩次腔體，抽到1e-1 torr

    4.關閉抽氣閥

    5.調整氣壓使腔體上的指針降至上方黑線(氣壓差太低會使腔體漏氣)，靜至1小時

  7. 回溫:


    8. 1.把"手"按掉，取消HOLD，等40分

    2.將真空閥慢慢轉開，洩完氣後，即可將樣品取出

    3.蓋上真空閥，將腔體氣壓抽至1e-1 torr

  8. 確認YBCO是否成功鍍上:


    9. 1.將樣品置於量測基板凹槽內，以單芯線連接樣品四個角落和量測基板上，兩兩平行(〓█〓)

    2.將樣品裝入四點測量管後，接上機械幫浦，並將氮氣通入四點測量管，使管中皆為氮氣

    3.將四點測量管插入液態氮中，並將V+、V-、I+、I-和溫度計的線接上

    4.打開Labview中的"R-T最新版"的檔案

    5.打開溫控儀、電流供應器和奈伏計
